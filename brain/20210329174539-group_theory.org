* Definitions
- <<<Group>>> :: generalizes the concept of ordinary addition
  + associative property :: $(a * b) * c = a * (b * c)$
  + Identity Element :: $a * e = e * a = a$
  + Inverse :: $(a*a')=(a'*a)=e$
- <<<Abelian>>> Group or <<<Commutative>>> Group :: if operation is commutative $(a*b)=(b*a)$
