* Creating shared libraries
1. First create object with [[file:20210115201710-position_independent_code.org][Position Independent Code]]
#+begin_src bash
gcc -fPIC -g -c -Wall a.c
#+end_src

2. then create shared library
#+begin_src bash
gcc -shared -o liba.so a.o
#+end_src
